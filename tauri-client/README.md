# YsnTrans Client

This directory contains the Tauri 2 + React + TypeScript desktop client for YsnTrans Screenshot Translator.

- Main project README: [../README.md](../README.md)
- 中文文档: [../README.zh-CN.md](../README.zh-CN.md)

## Development

```bash
npm install
npm run tauri dev
```

## Build

```bash
npm run build
npm run tauri build
```
