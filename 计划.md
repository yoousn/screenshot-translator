# v1.1 截图态区域录屏与滚动截图修复计划

## Summary
- 移除左侧独立“录屏”页面；录屏只在截图状态内操作。
- `模型/视频配置` 继续负责 OCR、FFmpeg 下载/更新、路径选择和运行包管理。
- 新录屏流程：`Alt+A` → 框选区域 → 点击录像按钮 → 红色录制框 + 录制工具栏 → 开始/暂停/继续/完成 → 选择保存 mp4。
- 滚动截图改成不退出截图状态、不强制抢鼠标，优先采用手动滚动采集。

## Key Changes
- **导航与页面**
  - 从左侧导航删除“录屏”入口。
  - 废弃独立 `Recording.tsx` 页面，不再提供全屏录屏配置页。
  - FFmpeg 相关配置保留在“模型/视频配置”：检测、选择、官方 GitHub 下载/更新、音频设备状态说明。

- **截图态录屏入口**
  - 在截图工具栏增加“录像”按钮。
  - 点击后进入录屏准备模式：选区边框变红，下方工具栏切换为录屏相关按钮。
  - 录屏设置包含：`30/60 FPS`、`480P/720P/1080P/原画`，默认 `1080P`。
  - 音频包含：`静音`、`麦克风`、`系统声音`、`系统声音 + 麦克风`。
  - 只有检测到可用系统声音设备时才允许选择系统声音。

- **录屏交互**
  - 点击“开始”后开始录制选区。
  - 开始按钮左侧显示“暂停/继续”。
  - 录制中原“开始”按钮变成“完成”。
  - 工具栏录制时不消失，保持可暂停、继续、完成。
  - `Ctrl+S` 等同点击“完成”。
  - `Esc` 取消录制并清理临时文件。
  - `Alt+F4` 强制退出所有截图/录屏状态，并确保不残留 FFmpeg 进程。

- **区域录屏后端**
  - 录制区域来自当前截图选区的物理坐标：`x/y/w/h`。
  - FFmpeg 使用 Windows `gdigrab` 区域参数录制选区，而不是全屏。
  - 暂停/继续采用“分段录制”：暂停时结束当前片段，继续时开始新片段，完成时 concat 合并为一个 mp4。
  - 完成后弹出保存对话框，默认文件名：`YSN_Recording_YYYY-MM-DD_HH-mm-ss.mp4`。
  - 用户取消保存时提示保留临时文件，可重试保存或放弃。
  - 尝试对截图辅助窗口设置录制排除，避免红框和工具栏进入视频；若系统不支持则提示可能被录入。

- **滚动截图修复**
  - 点击 `SCR` 后不再直接退出截图状态。
  - 进入滚动截图模式：选区边框变橙色，工具栏切换为“开始采集 / 完成 / 取消”。
  - 默认采用手动滚动采集：用户自己滚动目标窗口，程序定时采集选区帧。
  - 不再强制移动鼠标或自动滚轮。
  - 完成后拼接长图并复制/发送结果。
  - 失败或取消时恢复原截图状态和选区，不关闭截图窗口。
  - 保留现有重叠匹配算法，并增加无变化帧跳过、最大帧数、超时、取消清理保护。

## Test Plan
- **构建测试**
  - `npm run build`
  - `cargo test --manifest-path tauri-client/src-tauri/Cargo.toml`
  - `python -m pytest server`
  - `git diff --check`
  - `.\build.bat`

- **录屏手测**
  - `Alt+A` 后框选区域，点击录像，确认边框变红且工具栏切换。
  - 30 FPS / 1080P / 静音录制，完成后选择保存，mp4 可播放且只包含选区。
  - 60 FPS / 原画录制，确认输出分辨率和帧率正确。
  - 麦克风录制成功。
  - 系统声音仅在检测到 WASAPI / Stereo Mix / 虚拟声卡时可选。
  - 暂停后画面变化不进入最终视频；继续后新片段能正常合并。
  - `Ctrl+S` 完成保存。
  - `Esc` 取消录制且不残留 FFmpeg。
  - `Alt+F4` 强制退出且不锁死。

- **滚动截图手测**
  - 点击 `SCR` 不退出截图状态。
  - 手动滚动采集不抢鼠标。
  - 完成后输出长图。
  - 失败/取消后仍回到原截图选区。
  - 连续多次截图、录屏、滚动截图切换不残留窗口或进程。

## Assumptions
- 录屏入口只保留在截图状态工具栏。
- 独立“录屏”页面删除。
- `模型/视频配置` 是唯一 FFmpeg/OCR 运行包管理入口。
- 区域录屏默认分辨率为 `1080P`。
- 系统声音能力取决于当前 Windows 设备和 FFmpeg 支持，不强行假装可用。
- 滚动截图优先改成手动滚动采集，解决当前自动滚动抢鼠标和退出截图状态的问题。
