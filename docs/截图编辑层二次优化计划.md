# 截图编辑层二次优化计划

## 目标
- 修复截图编辑工具栏中文显示为 `\\uXXXX` 或乱码的问题。
- 优化工具栏图标大小与红叉/绿勾观感。
- 改善文字标注输入、编辑、拖动体验。
- 限定标注拖动范围：只允许方框、圆、文字拖动；马赛克、笔、箭头画错直接撤销。
- 修复翻译 loading 在扁平选区中被压扁的问题。
- 确认 `Alt+A` 全局截图优先级和构建图标来源。

## 实现范围
- 主要修改 `tauri-client/src/pages/ScreenshotPage.tsx`。
- 如图标缓存导致产物仍显示旧图标，清理 release 构建缓存并重新构建。
- 不引入重型绘图库，不改 OCR/翻译后端流程。

## 交互规则
- 工具栏按钮统一使用图标和 tooltip，按钮约 36x36。
- 文字工具点击选区后显示居中的浮动输入框；Enter 确认，Esc 取消。
- 点击已有文字标注可再次编辑；拖动已有文字可移动。
- 方框、圆支持整体拖动。
- 马赛克、笔、箭头不支持拖动，只支持撤销删除。
- 画笔、方框、圆、箭头、文字显示大小和颜色；马赛克只显示大小。

## 验证
- `npm run build`
- `cargo check`
- `npx tauri build --no-bundle`
- 生成 `D:\Desktop\自制截图\tauri-client-new.exe`。
