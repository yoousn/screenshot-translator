# YSN 客户端原生功能全面恢复与 ACL 修复实施计划

本计划旨在解决当前客户端存在的 window 权限控制报错问题，并全面恢复与实现 MVP 定义的全部截图、翻译、贴图、OCR 及辅助截图交互功能。

## 1. 待修复与恢复的功能列表

1. **修复 ACL 权限报错 (P0)**：
   - 解决前端 JS 调用 `WebviewWindow.show()` / `hide()` 时报 `Command plugin:window|show not allowed by ACL` 错导致无法退出/隐藏窗口的问题。
2. **恢复“截图翻译” (Alt + T / Ctrl + Q) (P0)**：
   - 支持 `Alt + T` 直接进入翻译截图模式。
   - 截图框选后，自动（或手动按 `Ctrl+Q` / 点击悬浮条“翻译”按钮）调用后端 `/api/translate` 接口。
   - 翻译成功后，**直接在截图原位置擦除旧文字并重绘中文译文**，用户可在画布上直接预览翻译结果，并一键“复制”或“保存”处理后的图片。
3. **恢复“延迟截图” (P1)**：
   - 倒计时 3 秒，UI 显示大字倒计时，结束后自动隐藏主窗口并调起截图。
4. **恢复“固定到屏幕 (Pin/贴图)” (P1)**：
   - 框选后点击“贴图”，动态创建一个独立的无边框、置顶、半透明小窗口显示截图，支持鼠标拖动移动、鼠标滚轮缩放、Esc/双击销毁。
5. **恢复“文本识别 (OCR)” (P1)**：
   - 框选后点击“识字”，调用后端 `/api/ocr` 识别选区文字，自动将文本复制到剪贴板。
6. **恢复“截取全屏”与“活动窗口截图” (P2)**：
   - 截取当前完整画面或获取焦点窗口范围，快速写入剪贴板。

## 2. 详细技术实现方案

### 2.1 修复 ACL 权限与主窗口关闭

#### [MODIFY] [default.json](file:///c:/Users/ysn/Desktop/zzjt/tauri-client/src-tauri/capabilities/default.json)
在 `permissions` 中添加 `"core:window:default"` 与 `"core:event:default"`。Tauri v2 默认沙箱会拦截所有未授权的前端 Webview IPC 调用，引入该权限集后前端即可正常调用 `.show()`、`.hide()`、`.setFocus()` 等窗口控制命令。

```json
  "permissions": [
    "core:default",
    "opener:default",
    "global-shortcut:default",
    "core:window:default",
    "core:event:default"
  ]
```

### 2.3 恢复截图翻译与原地重绘 (In-Place Translation)

#### [MODIFY] [ScreenshotPage.tsx](file:///c:/Users/ysn/Desktop/zzjt/tauri-client/src/pages/ScreenshotPage.tsx)
- **快捷键绑定**：Canvas 监听 `Ctrl + Q` 快捷键。
- **悬浮菜单扩展**：在截图浮动条中加入 **“翻译 (Ctrl+Q)”**、**“贴图 (Pin)”**、**“识字 (OCR)”** 按钮。
- **重绘工作流**：
  1. 用户框选区域后，点击“翻译”或触发自动翻译。
  2. 调用 `capture_region` 获取选区 Base64 图像。
  3. 请求后端 `${serverUrl}/api/translate` 接口，展示“正在重绘译文...” loading 动画。
  4. 获取翻译后图像数据，将其渲染并叠加覆盖在 Canvas 选区矩形内部。
  5. 用户预览满意后，点击“复制”或“完成”，将**替换译文后的新图像**写入系统剪贴板。

## 3. 验证与测试计划

### 自动化与编译测试
- 运行前端编译检查：`npm run build`
- 运行后端编译检查：`$env:PATH = "$env:USERPROFILE\.cargo\bin;" + $env:PATH; cargo check`
- 生成正式 Release 程序：`npx tauri build --no-bundle`

### 手动功能验收
1. **ACL 验证**：启动后，截图框选并按 `Esc` 或右键，验证是否不再抛出 `ACL` 错误，且能顺利隐藏退出。
2. **截图翻译验证**：框选英文区域，按 `Ctrl+Q` 翻译，验证译文是否自动重绘并替换原位置上的英文；复制后粘贴到微信中验证是否为中文图片。
3. **贴图验证**：框选区域点击“贴图”，验证桌面上是否出现独立悬浮图层，且支持拖动、滚轮缩放、Esc/双击退出。
4. **延迟截图验证**：点击延迟截图，验证屏幕是否有“3, 2, 1”倒计时，并在倒计时结束时调起截图窗口。
